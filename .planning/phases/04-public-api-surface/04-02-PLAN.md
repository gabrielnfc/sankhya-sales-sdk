    - gatewayCall with options.timeout uses custom timeout
    - requestWithRetry without options uses this.timeout (default behavior preserved)
    - options.idempotencyKey adds X-Idempotency-Key header to request
  </behavior>
  <action>
    1. Add import at top of src/core/http.ts:
       `import type { RequestOptions } from '../types/config.js';`

    2. Update all 4 public method signatures to accept optional `options?: RequestOptions` as last param:
       - `restGet<T>(path: string, params?: Record<string, string>, options?: RequestOptions): Promise<T>`
       - `restPost<T>(path: string, body: unknown, options?: RequestOptions): Promise<T>`
       - `restPut<T>(path: string, body: unknown, options?: RequestOptions): Promise<T>`
       - `gatewayCall<T>(modulo: string, serviceName: string, requestBody: Record<string, unknown>, options?: RequestOptions): Promise<T>`

    3. Pass `options` through to `requestWithRetry`:
       - Update requestWithRetry signature: add `options?: RequestOptions` after `isRetry`
       - Each public method passes `options` to requestWithRetry

    4. In requestWithRetry, implement per-call timeout:
       ```typescript
       const timeoutMs = options?.timeout ?? this.timeout;
       const internalController = new AbortController();
       const timeoutId = setTimeout(() => internalController.abort(), timeoutMs);

       const signals: AbortSignal[] = [internalController.signal];
       if (options?.signal) signals.push(options.signal);
       const combinedSignal = signals.length === 1
         ? signals[0]
         : AbortSignal.any(signals);
       ```
       Use `combinedSignal` in fetch call instead of `controller.signal`.

    5. In requestWithRetry, add idempotency header:
       ```typescript
       if (options?.idempotencyKey) {
         headers['X-Idempotency-Key'] = options.idempotencyKey;
       }
       ```

    6. On 401 retry, pass `options` through: `return this.requestWithRetry<T>(url, method, path, body, true, options);`

    7. Add tests in tests/core/http.test.ts for:
       - Custom timeout value used (mock setTimeout or check AbortController)
       - External signal abort throws TimeoutError
       - Idempotency key header forwarded
       - Default timeout used when no options
