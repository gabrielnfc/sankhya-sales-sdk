Output: Updated pedidos.ts and financeiros.ts with RequestOptions on mutations, plus unit tests.

       `import type { RequestOptions } from '../types/config.js';`

       - `criar(pedido: PedidoVendaInput, options?: RequestOptions)`
       - `atualizar(codigoPedido: number, pedido: PedidoVendaInput, options?: RequestOptions)`

       - `this.http.gatewayCall('mgecom', 'SelecaoDocumentoSP.faturar', {...}, options)` (faturar)
       - `this.http.gatewayCall('mgecom', 'CACSP.incluirNota', {...}, options)` (incluirNotaGateway)
       - `this.http.gatewayCall('mgecom', 'CACSP.incluirAlterarItemNota', {...}, options)` (incluirAlterarItem)
       - `this.http.gatewayCall('mgecom', 'CACSP.excluirItemNota', {...}, options)` (excluirItem)

    4. Do NOT add options to `consultar()` or `consultarTodos()` -- these are read-only methods.
