**Step 1: Install the coverage provider.**

Run:
```bash
npm install -D @vitest/coverage-v8@^3.2.4
```

IMPORTANT: The version MUST match the installed vitest major.minor (3.2.x). Do NOT install latest (which may be 4.x). After install, verify with `npm ls @vitest/coverage-v8` that it resolves to 3.2.x.

**Step 2: Update vitest.config.ts.**

Replace the entire `export default defineConfig(...)` block with:

```typescript
export default defineConfig({
  test: {
    globals: true,
    environment: 'node',
    include: ['tests/**/*.test.ts'],
    testTimeout: 30_000,
    env: loadEnv(),
    coverage: {
      provider: 'v8',
      include: ['src/**/*.ts'],
      exclude: ['src/types/**/*.ts', 'src/index.ts'],
      thresholds: {
        lines: 90,
        functions: 90,
        branches: 90,
        statements: 90,
      },
    },
  },
});
```

This preserves all existing config (globals, environment, include, testTimeout, env) and adds the coverage block.

The `exclude` array removes:
- `src/types/**/*.ts` — type-only files with no runtime code
- `src/index.ts` — barrel re-export file with no logic

**Step 3: Verify the test:coverage script already exists in package.json.**

package.json already has `"test:coverage": "vitest run --coverage"` — no change needed there. Verify it works by running it.
